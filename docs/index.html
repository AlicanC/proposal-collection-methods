    2%Set%.isSet(argument)

    When the isSet method is called, the following steps are taken:

    1. If Type(argument) is not Object, return false.
    2. If argument is a Set exotic object return true.
    3. If argument does have a [[SetData]] internal slot return true.
    4. Return false.

    The length property of the isSet method is 1.

    3Set.prototype.filter( _callbackfn [ , thisArg ] )

    When the filter method is called, the following steps are taken:

    1. Let S be the this value.
    2. If Type(S) is not Object, throw a TypeError exception.
    3. If S does not have a [[SetData]] internal slot, throw a TypeError exception.
    4. If IsCallable(callbackfn) is false, throw a TypeError exception.
    5. If thisArg was supplied, let T be thisArg; else let T be undefined.
    6. Let C be ? Get(S, "constructor").
    7. If IsConstructor(C) is true, then
      1. Let A be ? Construct(C)
    8. Else,
      1. Throw a TypeError exception.
    9. Let adder be ? Get(A, "add")
    10. If IsCallable(adder) is false, throw a TypeError exception.
    11. Let entries be the List that is S.[[SetData]]
    12. For each e that is an element of entries, in original insertion order, do:
      1. If e is not empty, then
        1. Let selected be ToBoolean(? Call(callbackfn, T, « e, e, S »))
        2. If selected is true, then:
          1. Call(adder, S, « e »)
    13. Return A.

    8Set.prototype.addElements(...items)

    When the addElements method is called, the following steps are taken:

    1. Let len be the actual number of arguments passed to this function.
    2. Let items be the List of arguments passed to this function.
    3. Let C be the this value.
    4. Let k be 0.
    5. Let adder be ? Get(C, "add")
    6. If IsCallable(adder) is false, throw a TypeError exception.
    7. Repeat while k < len
      1. Let kValue be items[k].
      2. Call(adder, C, kValue)
      3. Increase k by 1.
    8. Return C.

    The length property of the addElements method is 0.

    10Set.prototype.union(...items)

    When the union method is called, the following steps are taken:

    1. Let len be the actual number of arguments passed to this function.
    2. Let items be the List of arguments passed to this function.
    3. Let S be the this value.
    4. Let k be 0.
    5. Let C be ? Get(S, "constructor").
    6. If IsConstructor(C) is true, then
      1. Let A be ? Construct(C, « _S »)
    7. Else,
      1. Throw a TypeError exception.
    8. Let adder be ? Get(A, "add").
    9. If IsCallable(adder) is false, throw a TypeError exception.
    10. Repeat while k < len
      1. Let iterable be items[k].
      2. Let iter be GetIterator(iterable).
      3. If iter is undefined, throw a TypeError exception.
      4. Repeat,
        1. Let next be IteratorStep(iter).
        2. If next is false, exit innermost loop.
        3. Let nextValue be ? IteratorValue(next).
        4. Let status be Call(adder, A, « nextValue.[[Value]] »)
        5. If status is an abrupt completion, return IteratorClose(iter, status)
      5. Increase k by 1.
    11. Return A.

    The length property of the union method is 1.
